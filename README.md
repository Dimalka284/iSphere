# iSphere - Premium Apple E-Commerce Experience

iSphere is a modern, high-end e-commerce platform built with Laravel, designed specifically for premium electronics retail. It features a stunning, minimalist aesthetic, a seamless shopping experience, and a powerful administrative suite.

## 🚀 Features

### **Storefront & Shopping**
- **Premium Design**: A high-end visual experience using glassmorphism, modern typography (Outfit & Inter), and subtle micro-animations.
- **Product Discovery**: Category-based browsing (iPhone, Mac, etc.) with detailed product showcases.
- **Smart Cart System**: Add to cart functionality with real-time updates and quantity management.
- **Instant Checkout**: "Buy Now" feature for a frictionless purchasing experience.
- **Secure Payments**: Integrated with **Stripe** for reliable and secure payment processing.

### **User Experience**
- **Social Authentication**: Quick login via Google using Laravel Socialite.
- **Secure Accounts**: Standard email/password authentication with protected routes.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices.

### **Administrative Suite (iSphere Admin)**
- **Business Insights**: Real-time revenue analytics and dashboard overview.
- **Product Management**: Full CRUD system to manage the product catalog, inventory, and variants.
- **Order Tracking**: Comprehensive order management system to monitor statuses and payments.
- **Customer Database**: Manage your customer base with detailed activity logs and join dates.

## 🛠️ Tech Stack

- **Backend**: [Laravel](https://laravel.com/)
- **Frontend**: [Tailwind CSS v4](https://tailwindcss.com/),
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Payments**: [Stripe](https://stripe.com/)
- **Authentication**: Laravel Socialite (Google)
- **Typography**: Google Fonts (Inter & Outfit)
